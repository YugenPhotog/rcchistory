## Essential Keyboard Shortcuts

### Navigation
| Key | Action |
|-----|--------|
| **→** (Right Arrow) | Next slide |
| **←** (Left Arrow) | Previous slide |
| **Space** | Next slide |
| **Shift + Space** | Previous slide |
| **Home** | First slide |
| **End** | Last slide |

### Presentation Modes
| Key | Action |
|-----|--------|
| **F** | **Fullscreen** (press again or ESC to exit) |
| **S** | **Speaker Notes View** (shows current slide, next slide, timer, notes) |
| **O** or **ESC** | **Overview Mode** (see all slides at once) |
| **B** or **.** | **Blackout** (pause with black screen) |

### Other Features
| Key | Action |
|-----|--------|
| **Alt + Click** | Zoom in on specific area |
| **?** | Show all keyboard shortcuts |

## Speaker Notes View (Press 'S')

When you press **S**, you'll see:
- **Left Panel:** Current slide (what students see)
- **Right Panel:** Next slide preview
- **Bottom Panel:** Your speaker notes with teaching points
- **Timer:** Elapsed time and current time

### Tips for Speaker View:
1. Use on your laptop while projecting main view to TV/screen
2. Notes include:
   - Key teaching points
   - Discussion questions to ask
   - Transition cues
   - Time estimates for sections
3. Keep speaker view on your laptop, students see main view on TV

## Troubleshooting

### Images Not Loading?
- Images are hosted on Netlify at `/images/` path
- Ensure internet connection active
- Check browser console for errors (F12)

### Slides Look Wrong?
- Make sure browser window is maximized
- Try different browser (Chrome recommended)
- Clear browser cache and reload (Ctrl+F5)

### Controls Not Working?
- Click on presentation first to focus it
- Make sure no dialog boxes are open
- Try refreshing page (F5)

## Best Practices

### Before Class:
1. ✓ Test presentation on classroom TV/projector
2. ✓ Open speaker view (S) on your laptop
3. ✓ Start in fullscreen mode (F)
4. ✓ Have presentation open 5 minutes early
5. ✓ Test internet connection (for images)

### During Class:
1. Use **Space** or **→** to advance slides
2. Refer to speaker notes for discussion questions
3. Use **B** to blackout during extended discussions
4. Use **O** to show overview if students request
5. **Alt+Click** to zoom in on specific slide elements

### After Class:
1. Press **ESC** to exit fullscreen
2. Check speaker view timer to note actual time spent
3. Review notes for next time
