### Reveal.js Theme Directive
Use the **Manifest Destiny (Cinematic Mode)** theme.

**Base:** Reveal.js built-in theme `black.css`

**CDN Reference:**
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/reveal.js@4.5.0/dist/theme/black.css">
```

**Custom Style Block (insert in <style> or inline CSS):**
```css
:root {
  --primary-bg: #1a1a2e;      /* deep navy background */
  --secondary-bg: #16213e;    /* darker gradient tone */
  --accent-color: #e94560;    /* crimson-rose accent */
  --text-color: #f1f1f1;      /* soft white text */
  --highlight-color: #ffd700; /* gold highlight */
}

.reveal {
  font-family: 'Helvetica Neue', Helvetica, Arial, sans-serif;
  background: var(--primary-bg);
  color: var(--text-color);
}

/* Headings */
.reveal h1 {
  color: var(--accent-color);
  font-size: 1.15em;
  text-transform: none;
  margin-bottom: 0.3em;
  line-height: 1.2;
}
.reveal h2 {
  color: var(--highlight-color);
  font-size: 0.95em;
}
.reveal h3 {
  color: var(--text-color);
  font-size: 0.85em;
}

/* Text */
.reveal p, .reveal li {
  font-size: 0.85em;
  line-height: 1.4;
  color: var(--text-color);
}
.reveal strong { color: var(--highlight-color); font-weight: bold; }
.reveal em { color: var(--accent-color); }

/* Quote boxes */
.quote-box {
  background: var(--secondary-bg);
  border-left: 5px solid var(--accent-color);
  padding: 1em;
  margin: 0.5em 0;
  border-radius: 5px;
  font-style: italic;
}
.quote-attribution {
  text-align: right;
  font-size: 0.85em;
  color: var(--highlight-color);
}

/* Image placeholders */
.image-placeholder {
  background: var(--secondary-bg);
  border: 3px dashed var(--accent-color);
  padding: 1.5em 1em;
  border-radius: 10px;
  text-align: center;
  color: var(--highlight-color);
}
.two-column {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 2em;
  align-items: start;
}
.section-title {
  background: linear-gradient(135deg, var(--secondary-bg) 0%, var(--primary-bg) 100%);
}
```

**Reveal.js Configuration (Cinematic Mode):**
```javascript
Reveal.initialize({
  hash: true,
  transition: 'convex',             // smooth 3D arc between slides
  backgroundTransition: 'zoom',     // cinematic camera zoom for background
  transitionSpeed: 'slow',          // slower pacing for dramatic rhythm
  controls: true,
  controlsLayout: 'bottom-right',
  progress: true,
  slideNumber: 'c/t',
  center: true,
  width: 1280,
  height: 720,
  margin: 0.04,
  minScale: 0.2,
  maxScale: 2.0
});
```

**Usage Example:**
> “Generate a Reveal.js lecture deck using my *Manifest Destiny (Cinematic Mode)* theme.  
> Use the dark navy + gold palette and cinematic transitions (`convex` + `zoom` + `slow`).  
> All slide notes should appear inside `<aside class="notes">` for presenter view.”
